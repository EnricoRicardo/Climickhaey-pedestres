# Climickhaey: Navegação Urbana Segura (MVP)

O **Climickhaey** é um sistema de navegação para pedestres no centro de São Paulo que utiliza a Teoria dos Grafos para calcular rotas otimizadas, equilibrando distância e fatores de risco urbano, com foco em acessibilidade.
